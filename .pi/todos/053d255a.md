# Speaker Notes Support

Completed speaker notes support with presentation API and notes files.

## Implementation
- Added `/_/api/notes` endpoint for notes markdown files with hash sanitization.
- Added speaker notes panel in presenter view.
- Presenter fetches notes on slide change with caching, loading, and empty states.
- Display script reports notes from `window.miniPresenter.getNotes` when available.
- Added demo notes in `examples/basic/notes/`.
