# Basic Presenter View

Create the presenter control interface with:
1. Navigation controls (prev/next buttons + keyboard)
2. Timer (elapsed time with pause/reset)
3. Connection status (number of displays)
4. Current slide indicator

## Layout (MVP)

```
┌──────────────────────────────────────────────────────────┐
│  mini-presenter    [00:15:32]    Current: #slide-5       │
├──────────────────────────────────────────────────────────┤
│                                                          │
│              (Current slide preview - iframe)            │
│                                                          │
├──────────────────────────────────────────────────────────┤
│  [◀ Prev]        [▶ Next]        [⏱ Reset]              │
│                                                          │
│  Displays connected: 2                                   │
└──────────────────────────────────────────────────────────┘
```

## Files

### client/presenter.html
- Clean, minimal UI
- Responsive layout
- Dark theme (easier on eyes during presentation)

### client/presenter.css
- Flexbox/grid layout
- Large, easy-to-click buttons
- Clear visual hierarchy

### client/presenter.js
- WebSocket connection (register as 'presenter')
- Send navigation commands
- Receive and display state updates
- Timer logic (start on first slide change, pause, reset)
- Keyboard shortcuts (arrow keys, space)

## Keyboard Shortcuts
- `→` / `Space` / `PageDown` - Next slide
- `←` / `PageUp` - Previous slide
- `Home` - First slide
- `End` - Last slide
- `T` - Toggle timer pause
- `R` - Reset timer
