# Project Setup

Create the initial project structure:

```
mini-presenter/
├── package.json
├── bin/
│   └── mini-presenter.js    # CLI entry point
├── src/
│   ├── server.js            # HTTP server (plain Node)
│   ├── websocket.js         # WebSocket handling
│   └── injector.js          # HTML injection logic
├── client/
│   ├── presenter.html
│   ├── presenter.css

│   └── injected.js
```
